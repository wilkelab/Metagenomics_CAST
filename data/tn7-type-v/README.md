# Data files for Type V Tn7 CASTs

  - `fig-5d-tnsc.fa` TnsC sequences for Figure 5D.
  - `fig-5e-tnsb.fa` TnsB sequences for Figure 5E.
  - `our-tnsb.fa` All TnsB sequences from Type V systems.
  - `our-tnsc.fa` All TnsC sequences from Type V systems.
  - `tn5053-tnsB.fasta` TniA (=TnsB) from Tn5053.
  - `tn5053-tnsC.fasta` TniB (=TnsC) from Tn5053.
